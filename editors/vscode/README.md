# Niki-docAI VS Code Extension

Integration for Niki-docAI Context Ops Toolchain.

## Features

- **Thinking Context**: Hover over any file to see the "Mental Context" (Rules, Memories, Architecture) that AI should know about.
- **LSP Integration**: Seamlessly connects to `ndoc server`.
- **Dogfooding**: Support for local development via `ndoc.server.localPath` setting.

## Configuration

- `ndoc.pythonPath`: Path to python interpreter.
- `ndoc.server.localPath`: [Dev Mode] Path to local `nk_doc_ai` source code for live updates.
